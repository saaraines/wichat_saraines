== Contexto y Alcance


=== Contexto Empresarial

El proyecto WIChat tiene como objetivo crear una versión en línea del concurso de preguntas y respuestas “Saber y Ganar" para RTVE. Deberá aparecer una pregunta con una imagen y 4 posibles respuestas para responder en un tiempo limitado




=== Contexto técnico

WIChat es una aplicación web compuesta por diversos módulos, con una arquitectura basada en un enfoque de microservicios, con integración de APIs para la generación de preguntas y LLM.

**Componentes principales:**

* API RESTful para gestionar autenticación, preguntas y estadísticas de los usuarios.

* Integración con una base de datos para almacenamiento de la información.

* Generación de preguntas desde Wikidata

* Sistema de pistas implementando un LLM

* Gestión de usuarios

* Despliegue y monitoreo con contenedores de Docker y herramientas de integración contínua de Github
